# 🚀 Team Onboarding - Quick Start Guide

## 🎯 Welcome to Blu Blog Gen Team!

This guide will get you up and running in **5 minutes** so you can start contributing immediately.

## ⚡ 5-Minute Setup

### **1. Get Repository Access**
- **Repository**: `https://github.com/blureonlabs/final_blog_gen`
- **Contact**: @blureonlabs to get collaborator access
- **Clone**: `git clone https://github.com/blureonlabs/final_blog_gen.git`

### **2. Quick Environment Setup**
```bash
cd final_blog_gen
npm install
cp env.template .env.local
# Edit .env.local with your Supabase credentials
npm run dev
```

### **3. Start Developing**
```bash
git checkout main
git pull origin main
git checkout -b feature/your-first-feature
# Make changes, commit, push, create PR!
```

## 📋 Quick Reference

### **Branch Naming**
- `feature/user-auth` - New features
- `bugfix/login-error` - Bug fixes
- `hotfix/critical-bug` - Urgent fixes

### **Commit Messages**
```
feat(auth): add OAuth login support
fix(api): resolve user data fetching issue
docs(readme): update installation instructions
```

### **PR Process**
1. Create feature branch
2. Make changes and test
3. Create Pull Request
4. Request review
5. Address feedback
6. Merge after approval
